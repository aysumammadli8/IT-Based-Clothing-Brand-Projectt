# 👕 IT based clothing brand

## 📌 Layihə haqqında
TechWear informasiya texnologiyalarına əsaslanan müasir geyim markası layihəsidir. Layihə HTML, CSS və JavaScript istifadə edilərək hazırlanmış çoxsəhifəli web sistemdir.

Layihənin məqsədi istifadəçilərə texnologiya ilə inteqrasiya olunmuş innovativ alış-veriş təcrübəsi təqdim etməkdir.

---

## 🎯 Layihənin məqsədi
- Süni intellekt əsaslı geyim tövsiyə sistemi yaratmaq
- Virtual geyim sınaq funksiyası hazırlamaq
- İstifadəçiyə öz geyimini dizayn etmək imkanı vermək
- Bədən ölçülərinə uyğun avtomatik seçim sistemi qurmaq

---

## 🧠 Əsas funksiyalar

### 🤖 AI Style Assistant
İstifadəçinin verdiyi məlumatlara əsasən uyğun geyim təklif edir.

### 👕 Virtual Try-On System
İstifadəçi seçdiyi geyimi virtual olaraq üzərində görə bilir.

### 🎨 Custom Design Tool
İstifadəçi öz geyimini dizayn edə bilir (rəng, yazı və s.).

### 📏 Body Scan System
İstifadəçinin bədən ölçülərini analiz edərək uyğun ölçü təklif edir.
